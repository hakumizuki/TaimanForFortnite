## アプリ概要 About this App
 Fortniteで1v1で練習したい人に向けて作られた掲示板マッチングアプリです。
 
 一人が募集を出し、もう一人がそれにエントリーすることで、お互いがマッチングします。
 マッチング後、お互いにIDを交換し、募集内容にそってゲームをプレイします。
